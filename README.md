# hub repositories sub command

Add the `repositories` sub command to [hub](https://github.com/github/hub).

## Installation

```bash
./configure --prefix=$HOME/local
make
make install
```

## Usage

```bash
hub repositories
#=> Output all repository names of your GitHub public repositories.
```
